# LaTeX 模板合集

这是一个精选的 LaTeX 模板库，包含适用于各种学术和专业场景的现代化、美观且功能完备的模板。

所有模板均已配置为完美支持 **XeLaTeX** 编译引擎，原生支持 UTF-8 编码和现代系统字体渲染（包括原生中文环境支持）。

[🇺🇸 English Readme](README.md)

## 包含的模板

### 1. 学术论文模板 (`Academic_Paper/`)
一个标准且专业的**双栏**学术论文模板，格式参考了顶会（如 IEEE / ACM）的标准布局。
- **特性**：双栏排版、多作者与机构的美观展示 (`authblk`)、现代图表格式 (`booktabs`) 以及 BibTeX 参考文献支持 (`natbib`)。
- **结构特点**：正文部分已拆分至独立的 `sections/` 文件夹（如引言、相关工作、方法、实验、结论），非常适合基于 Git 的多人团队协作，显著减少代码合并冲突。

### 2. 学术/课程报告模板 (`Academic_Report/`)
一个通用的、单栏 A4 页面报告模板，专为大学课程作业、项目总结和毕业论文设计。
- **特性**：清晰的层级结构、自定义页眉页脚 (`fancyhdr`)、代码块语法高亮 (`listings`) 以及完善的实验结果/公式排版。

### 3. 学术作业模板 (`Academic_Homework/`)
一个干净整洁的模板，专门用于提交数学、理论物理或算法分析等课后作业。
- **特性**：包含自定义且带有边框的“解答/Solution”环境 (`mdframed`)，自带常用的数学定理宏包，同时具备标准的学生与课程信息页眉。

### 4. LaTeX 语法练习沙盒 (`LaTeX_Practice/`)
一个完整的演示文档，展示了 LaTeX 最核心的排版语法。非常适合刚开始学习 LaTeX 的新手作为对照参考。
- **特性**：包含基础文本格式化、行内/行间公式排版、多行对齐公式、列表、三线表以及图片插入与引用的代码示例。

## 如何编译使用

强烈建议使用 **XeLaTeX** 引擎编译这些文档，以保证最佳的排版效果（特别是其已内嵌 `ctex` 宏包，配合 XeLaTeX 能获得最完美的中文支持，且不需要额外配置字体族）。

### 命令行编译
进入对应的模板目录执行：

```bash
xelatex main.tex
bibtex main      # 如果模板引用了 references.bib 参考文献库
xelatex main.tex
xelatex main.tex
```

### 编辑器推荐 (VSCode / TeXShop / TeXstudio)
请将你的默认编译器或 Recipe 设置为 **XeLaTeX**。所有的 `main.tex` 首行都加入了 `%!TEX TS-program = xelatex` 魔法注释，目前绝大多数现代 LaTeX 编辑器都能根据这行注释无头自动选择正确的编译引擎。
